# GPS Real-Time Tracker
* Description: GPS Real-Time Tracker
  - This script is a real-time GPS tracker that:
  - Resets the GPS hardware
  - Continuously reads GPS data
  - Displays location, speed, and altitude
  - Converts raw GPS data into human-readable info
  - Uses online reverse-geocoding only when needed
  - Works offline when internet is unavailable
  - Updates every 10 seconds
  - Cleans up properly when stopped
